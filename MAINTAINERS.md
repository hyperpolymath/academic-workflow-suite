## Becoming a Maintainer

Maintainers are selected based on sustained contributions and demonstrated expertise. The process:

1. **Sustained Contribution** (6+ months)
   - Regular code contributions
   - High-quality pull requests
   - Helpful code reviews
   - Community support

2. **Technical Expertise**
   - Deep knowledge of relevant components
   - Understanding of project architecture
   - Security and privacy awareness

3. **Community Involvement**
   - Participation in discussions
   - Mentoring other contributors
   - Adherence to Code of Conduct

4. **Nomination Process**
   - Existing maintainers nominate candidates
   - Private discussion among core team
   - Consensus decision (no vetoes)
   - Public announcement

## Maintainer Responsibilities

### Code Review
- Review PRs within 5 business days
- Provide constructive feedback
- Ensure tests pass and code quality standards met
- Verify security implications

### Release Management
- Follow semantic versioning
- Maintain CHANGELOG.md
- Test releases thoroughly
- Coordinate release announcements

### Security Response
- Monitor security@academic-workflow-suite.org
- Respond to vulnerability reports within 24-48 hours
- Coordinate patches and disclosures
- Maintain security documentation

### Community Leadership
- Enforce Code of Conduct
- Welcome new contributors
- Respond to issues and discussions
- Represent project professionally

## Stepping Down

Maintainers may step down at any time by:
1. Notifying other maintainers privately
2. Transferring active responsibilities
3. Moving to emeritus status
4. Public announcement (optional)

We thank all maintainers for their service!

## Tri-Perimeter Contribution Framework (TPCF)

This project follows TPCF Perimeter 2 (Trusted Contributors) governance:

- **Perimeter 1 (Core)**: Maintainers listed above
- **Perimeter 2 (Trusted)**: Contributors with 3+ merged PRs, demonstrated expertise
- **Perimeter 3 (Sandbox)**: Open community contributions (issues, discussions, documentation)

See [TPCF.md](TPCF.md) for complete framework details.
